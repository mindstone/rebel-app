# Rebel Feature Screenshots

Real-product screenshots of Rebel features, ready to embed in READMEs, docs, and
other GitHub-facing material. All captures use demo/fixture data only — no real
user content.

These are promoted copies from Mindstone's internal UI capture library
(internal repo path: `marketing-assets/source/ui-capture-library/manifest.md` —
not part of the public mirror). If the product UI has materially changed,
re-capture there first, then refresh the copy here.
