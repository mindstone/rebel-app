# Provision Cloud VM

Read the cloud VM setup guide at `docs/research/260507_cloud_droid_vm_setup.md` and walk the user through the provisioning process.

Key reference: the provisioning script is `scripts/cloud-vm-provision.sh`.

Help the user with:
- Choosing between personal vs shared VM setup
- Running the provisioning script with the right arguments
- Setting up Google Drive transcript upload (Section 12 of the guide)
- Troubleshooting any issues that come up during provisioning
- Verifying the setup works (SSH, tmux, droid-session, transcript upload)
